Create a new AWS Account

In this page you will learn how to sign up to Amazon Web Services (AWS) if you don't have any account yet.

Getting Started

To create a new AWS account, open your browser and access https://aws.amazon.com

Search at the top right-hand corner for Sign In to the Console and click on it.

In the next page, at the bottom, you will find an button to create a new account. Click on it:

Now you need to type the email for the root user (in this case, it will be yourself. You may use your umich.edu account) and an account name (you may change it later). Once you typed all information, press Verify email address.

AWS will send an email with a 6-digit verification code to confirm you have access to the provided email. This code is valid for 10 minutes. Copy this code and paste it in the textfield and then press Verify:

If the information provided is correct, you will start a 5-step procedure before finishing your account creation.

Step 1: Create Password:

In this step you need to type your password and then press the Continue button.

Step 2: Add Contact Information

Select Personal as you will learn how to use this service. If you are creating an account for your future company/startup, you should select Business.

Add all information about yourself and don't forget to click on I have read and agree to the terms of the before press the Continue buton.

Step 3: Add Billing Information

Add all information related to billing. As AWS states, you can't proceed with the sign-up process until you add a valid payment method.

AWS will not charge you for usage below AWS Free Tier limits. They may temporarily hold up to $1 USD (or an equivalent amount in local currency) as a pending transaction for 3-5 days to verify your identity.

Step 4: Verify your phone number

In this step, AWS will ask you to confirm your phone number by using an PIN number.

Step 5: Choose your plan

In this step, select the AWS Free Tier. If everyting went well, you account will be activated within a few minutes. However, AWS says that the process can take up to 24 hours.

Once you account is fully activated, you will receive an confirmation email. Don't forget to check your spam folder for the confirmation email.
